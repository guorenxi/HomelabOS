# Usage
<!-- usage -->
```sh-session
$ npm install -g hlos
$ hlos COMMAND
running command...
$ hlos (-v|--version|version)
hlos/0.0.0 darwin-x64 node-v13.1.0
$ hlos --help [COMMAND]
USAGE
  $ hlos COMMAND
...
```
